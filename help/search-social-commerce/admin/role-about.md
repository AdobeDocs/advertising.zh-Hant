# 關於角色

角色決定使用者型別可以執行的許可權。 每個使用者都會獲派一個角色，並可執行該角色允許的所有活動。

只有具有[!DNL Adobe]帳戶管理員和管理員角色的使用者才能指派角色給使用者。
