# 檢視其他廣告商的資料

*僅管理多個廣告商帳戶的使用者*

## （新UI）檢視其他廣告商的資料

* 在資料表格上方的工具列中，選取廣告商。

  ![工具列中的廣告商選擇器](/help/search-social-commerce/assets/advertiser-selector.png "工具列中的廣告商選擇器")

## （舊版UI）檢視其他廣告商的資料

*僅限代理商帳戶管理員、[!DNL Adobe]帳戶管理員和系統管理員使用者角色*

* 在任何頁面右上角的廣告商清單中，選取廣告商。
