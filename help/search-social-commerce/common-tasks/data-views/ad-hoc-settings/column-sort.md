# 重新排序欄

<!-- The same in new UI and legacy CM views -->

1. （可用時）在欄標題的右側，按一下![向下箭頭](/help/search-social-commerce/assets/arrow-down-expand.png "向下箭頭")。

1. 選取排序選項：

   * *遞增排序* （A到Z，或1到10）

   * *遞減排序* （Z到A，或10到1）
