# 重新排序欄

1. （可用時）在欄標題的右側，按一下 ![向下鍵](/help/search-social-commerce/assets/arrow-down-expand.png "向下鍵").

1. 選取排序選項：

   * *升序排序* （A至Z，或1至10）

   * *降序排序* （Z至A，或10至1）
