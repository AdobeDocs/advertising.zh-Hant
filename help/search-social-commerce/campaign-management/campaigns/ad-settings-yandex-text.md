# [!DNL Yandex] 文字廣告設定

## [!UICONTROL Text Ad]

**[!UICONTROL Ad Title]：** 橫幅（廣告）的標題。 長度上限為33個字元，單一字元不能超過23個字元。

>[!NOTE]
>
>變更的廣告文案 [!DNL Yandex] ad會刪除現有廣告，並建立具有相同屬性的新廣告。

**[!UICONTROL Description Line 1]：** 橫幅（廣告）的正文。 長度上限為75個字元，一個字最多只能有22個字元。

>[!NOTE]
>
>變更的廣告文案 [!DNL Yandex] ad會刪除現有廣告，並建立具有相同屬性的新廣告。
