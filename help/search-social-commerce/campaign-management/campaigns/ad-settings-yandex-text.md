# [!DNL Yandex]文字廣告設定

## [!UICONTROL Text Ad]

**[!UICONTROL Ad Title]：**&#x200B;橫幅（廣告）的標題。 長度上限為33個字元，單一字元不能超過23個字元。

>[!NOTE]
>
>變更[!DNL Yandex]廣告的廣告復本會刪除現有廣告，並建立具有相同屬性的新廣告。

**[!UICONTROL Description Line 1]：**&#x200B;橫幅（廣告）的正文。 長度上限為75個字元，單一字元不得超過22個字元。

>[!NOTE]
>
>變更[!DNL Yandex]廣告的廣告復本會刪除現有廣告，並建立具有相同屬性的新廣告。
