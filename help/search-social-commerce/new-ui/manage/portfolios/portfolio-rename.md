# （新UI）重新命名投資組合

*Beta功能*

快速重新命名投資組合，無需開啟完整的投資組合設定。

1. 在主功能表中，按一下&#x200B;**[!UICONTROL Manage]>[!UICONTROL Portfolios]**。

1. 將游標停留在投資組合列上，然後按一下&#x200B;**[!UICONTROL ...]>[!UICONTROL Rename]**。

1. 編輯名稱，然後按一下&#x200B;**[!UICONTROL Apply]**。
