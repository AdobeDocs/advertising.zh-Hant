# （新UI）關於[!UICONTROL Ad Groups]檢視

*Beta功能*

[!UICONTROL Manage] > [!UICONTROL Ad Groups]檢視會列出所選廣告商帳戶之篩選檢視中的所有廣告群組。

## 可用動作

<!--

* 從列中刪除或變更廣告群組的狀態。
