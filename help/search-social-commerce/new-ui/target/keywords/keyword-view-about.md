# （新UI）關於[!UICONTROL Keywords]檢視

*Beta功能*

[!UICONTROL Manage] > [!UICONTROL Keywords]檢視包含兩個標籤：

* [!UICONTROL Keywords]索引標籤會針對選取的廣告商帳戶，列出篩選檢視中的所有關鍵字。

* [!UICONTROL Negatives]索引標籤會針對選取的廣告商帳戶，列出篩選檢視中的所有負面關鍵字。

## 可用動作

* [將限制指派給關鍵字，並從關鍵字取消指派限制](/help/search-social-commerce/new-ui/target/keywords/keyword-assignments-manage.md)
