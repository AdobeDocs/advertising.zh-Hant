# （新UI）關於[!UICONTROL Placements]檢視

*Beta功能*

[!UICONTROL Manage] > [!UICONTROL Placements]檢視包含兩個標籤：

* [!UICONTROL Placements]索引標籤會針對選取的廣告商帳戶，列出篩選檢視中的所有版位。

* [!UICONTROL Negatives]索引標籤會針對選取的廣告商帳戶，列出篩選檢視中的所有負面版位。

## 可用動作
