# （新UI）刪除已上傳的大量工作表和錯誤檔案

您可以手動刪除大量表單檔案、登陸頁面驗證錯誤檔案和其他錯誤檔案。 檔案會在上傳或產生30天後自動刪除。

>[!TIP]
>
>如果您張貼大量表單但收到錯誤，請勿刪除錯誤檔案，直到解決所有錯誤為止。

1. 在主功能表中，按一下&#x200B;**[!UICONTROL Setup]** \> **[!UICONTROL Bulksheets]**。

1. 選取要刪除的每個檔案旁的核取方塊。

1. 在大量動作工具列中按一下&#x200B;**[!UICONTROL Delete]**。

1. 在確認訊息中，按一下&#x200B;**[!UICONTROL Confirm]**&#x200B;以確認刪除。

   如果有任何選取的Bulksheet已排定發佈，您會在確認前收到通知。 刪除已排程的大量表單會取消擱置的貼文。
