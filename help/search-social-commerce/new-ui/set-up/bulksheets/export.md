# （新UI）在本機匯出產生或上傳的大量工作表檔案

您可以下載任何產生或上傳的Bulksheet檔案，包括錯誤檔案和登陸頁面驗證檔案，為期30天。 檔案會在上傳或產生30天後自動刪除。

1. 在主功能表中，按一下&#x200B;**[!UICONTROL Setup]** \> **[!UICONTROL Bulksheets]**。

1. （可選）使用大量表單清單上方的搜尋方塊，以篩選顯示的檔案。

1. 在Bulksheet清單中，按一下檔案的名稱。

1. 依照瀏覽器的正常程式開啟或儲存檔案。

   如需詳細資訊，請參閱瀏覽器的線上說明。
