# 關於[!UICONTROL Advertising Insights]

[!UICONTROL Advertising Insights]呈現有關您最佳化和作用中產品組合（包含作用中行銷活動）的視覺化、可操作的資料。

每個分析都會隨選產生，而輸出是可以下載的檔案或熒幕報告。 您可以刪除分析檔案的已產生例項。

## [!UICONTROL Advertising Insights]的型別

| Insight | 說明 |
| --- | --- |
| [!UICONTROL AMO-AA Tracking Discrepancy] | (僅具有Adobe Advertising-Adobe Analytics整合的廣告商)會識別追蹤問題，其特徵為天數的點按執行個體比率超出正常變數，或變數超過20%。 報表包含：<ul><li>包含資料差異摘要的[!DNL Microsoft PowerPoint]檔案；顯示具有最多追蹤問題之行銷活動的各行銷活動差異的表格和趨勢圖表；疑難排解建議；以及報告方法的說明。</li><li>[!DNL Microsoft Excel]檔案，其中包含產品組合中每個行銷活動的摘要資料，以及每個行銷活動過去月份的每日資料。</li></ul> |
| [!UICONTROL Attribution Analysis] | （[!DNL PowerPoint]簡報格式）表示不同的歸因模型何時可以改善單一投資組合的收入模型和最佳化。 |
| [!UICONTROL Campaign Caps] | （[!DNL PowerPoint]簡報格式）指出單一投資組合過去30天的支出是否受到行銷活動預算上限的限制，並建議調整投資組合設定以實現最佳投資報酬率。 |
| [!UICONTROL Day of Week] | （[!DNL PowerPoint]簡報格式）指出單一投資組合在過去30天內(DOW)當天的績效，並建議DOW支出目標以增加您的投資報酬率。<br><br>**注意：**&#x200B;如果訪客沒有足夠的支出，或過去兩天無法支出Portfolio，將無法用於深入分析。 |
| [!UICONTROL Delayed Revenue] | 測量產品組合的轉換延遲（從廣告點選到後續轉換的經過時間），並顯示由於延遲而導致的加權收入（現在稱為&quot;[目標值](/help/search-social-commerce/glossary.md#o-p)&quot;）、ROI和模型準確性方面的任何差異。 此分析可讓您偵測點按日期歸因的績效量度中最近的變更，並透過點按日期與交易日期來檢視報表的影響。<br><br>分析包含[!DNL Excel]個活頁簿，其中包含加權收入（現在稱為「目標值」）、收入正確性及延遲因子的個別工作表。 它也包含包含包含各種圖表的[!DNL Powerpoint]檔案。 |
| [!UICONTROL Event Path] | 識別不同的管道和行銷活動如何透過分析事件路徑和多點觸控轉換來推動轉換。 您可以上傳XLSX和ZIP （壓縮XLSX）格式的檔案，然後a)擷取事件或b)分析分類事件。 |
| [!UICONTROL Google Account Audit] | （適用於[!DNL Google Ads]帳戶；對預售最有用）提供帳戶績效的概觀，並著重說明不足之處，以說明管理帳戶的有效性。 稽核包括有關競標、比對型別、預算上限、每週績效天數、行動效能等資訊。 若要產生深入分析，您必須從[!DNL Google Ads]網頁使用者介面上傳1) [!DNL Google Ads]促銷活動、關鍵字，並變更帳戶的歷程記錄報告，以及b)從[!DNL Google Ads Editor]應用程式上傳帳戶的批次工作表檔案。 所有檔案都必須是CSV、TSV、TXT或ZIP （壓縮的CSV、TSV或TXT）格式。<br><br>此深入分析可用於協助潛在客戶瞭解Search、Social和Commerce如何提升效能並解決分析所強調的缺陷。 Adobe客戶團隊也可以使用該見解在正式上線客戶之前評估帳戶。 |
| [!UICONTROL Impression Share Lost] | （[!DNL PowerPoint]簡報格式）指出產品組合的預算已限制[!DNL Google Ads]個行銷活動的曝光比重，並據此建議變更預算和行銷活動&quot;[!UICONTROL Multiple]&quot;設定。 |
| [!UICONTROL Location Target Performance] | （針對[!DNL Google Ads]個行銷活動；[!DNL Excel]試算表格式）單一產品組合中每個位置目標的競標調整、點按數、成本和加權收入（現在稱為「[目標值](/help/search-social-commerce/glossary.md#o-p)」）。 資料適用於過去90天內的任何日期範圍，且您可以檢視每日結果或每個行銷活動的摘要。 |
| [!UICONTROL Match Type] | 過去90天內，單一投資組合中各種相符型別的成本和加權收入（現在稱為「[目標值](/help/search-social-commerce/glossary.md#o-p)」）的熒幕圖表。 |
| [!UICONTROL Mobile Optimization] | （[!DNL PowerPoint]簡報格式）依裝置型別顯示過去30天內單一產品組合的效能、評估行動裝置特定最佳化功能的目前設定，並建議調整設定。 |
| [!UICONTROL Model Accuracy] | 識別搜尋、Social和Commerce預測單一作用中或最佳化產品組合的支出和收入程度，並識別模型不正確的來源。 報表包含：<ul><li>[!DNL PowerPoint]檔案，包含過去七天之預測與實際成本、點按次數、每次點按成本(CPC)、收入、每次點按收入(RPC)及投資報酬率(ROI)的摘要；上個月的超支及支出不足競標單位彙總資料；趨勢圖，顯示上個月的成本模型準確度、收入模型準確度、預測成本、點按次數、CPC、收入、RPC及投資報酬率(ROI)與實際值的比較；點按層級區分的模型準確度；支出不足或超支的最高競標單位；以及改善模型準確度的建議。</li><li>[!DNL Excel]檔案，具有每個關鍵字的模型正確性及依日期的模型正確性。</li></ul>此深入分析提供比[!UICONTROL Portfolios]檢視中提供的產品組合層級模型準確度報告更多的詳細資料。 |
| [!UICONTROL Normalized Sim (Combined)] | 針對多個投資組合的指定目標支出層級（步驟），傳回最佳支出配置。 此深入分析會根據指定時段內的每週模擬，使用標準化的加權收入（現在稱為「[目標值](/help/search-social-commerce/glossary.md#o-p)」）。 您可以產生a)每個所選投資組合的個別模擬，或b)具有相同目標和貨幣的多個投資組合的單一模擬。 模擬會以試算表顯示。<br><br>針對單一、多產品組合的模擬，試算表包含兩張工作表：1)趨勢圖，顯示每個目標支出層級（步驟）的標準化加權收入（現在稱為「[目標值](/help/search-social-commerce/glossary.md#o-p)」），以及2)表格，顯示每個適用支出層級（步驟）的一列。 針對個別產品組合模擬，試算表包含1)列示所有適用產品組合和時間週期的摘要工作表，以及2)每個產品組合的兩個工作表： a)顯示每個目標支出層級（步驟）的標準化加權收入（現在稱為「[目標值](/help/search-social-commerce/glossary.md#o-p)」）的趨勢圖，以及b)顯示每個適用支出層級（步驟）一列的表格。 |
| [!UICONTROL Portfolio Launch] | （[!DNL PowerPoint]簡報格式）識別產品組合是否已準備好啟動（已變更為「最佳化」狀態）。 此報表包含預測績效與實際績效的比較；建議的產品組合開始日期以及應排除在成本與收入預測之外的日期清單；成本、收入、每個成本的收入及隨時間轉換的關鍵字；有關過去兩週支出上限的任何行銷活動的資訊；以及啟動或不啟動的建議。 |
| [!UICONTROL Quality Score] | 顯示過去90天內曝光次數介於[!DNL Google Ads]或[!DNL Microsoft Advertising]的關鍵字品質分數趨勢。 它包含a) [!DNL PowerPoint]檔案和b) [!DNL Excel]檔案，其中包含支出最高的100個關鍵字和（適用時）品質分數較差的前100個關鍵字。<br><br>**附註：**<ul><li>當產品組合有超過20,000個競標單位時，僅納入在過去30天（而非90天）內點選多次的關鍵字。</li><li>含有許多關鍵字資料的深入分析需要較長時間才能產生。</li></ul> |
| [!UICONTROL Query Cross Matching] | （[!DNL Google Ads]帳戶）識別[!DNL Google Ads]符合多個關鍵字的搜尋查詢執行個體，並提供將流量導向到何處的建議。 使用這些建議來改善關鍵字效能，並因此改善模型準確性。<br><br>此分析包括交叉比對查詢、比對關鍵字和效能資料的試算表；網路圖表可協助您視覺化交叉比對查詢和關鍵字之間的關係；基礎搜尋查詢報表；以及如何解譯資料的資訊。 |
| [!UICONTROL Settings Audit] | 指出您的產品組合是否根據最佳實務進行設定。 內容包括：<ul><li>[!DNL PowerPoint]檔案，顯示是否根據最佳實務設定您所有產品組合及其元件行銷活動的關鍵設定、是否有任何行銷活動未指派給產品組合，以及偏離最佳實務的後果。</li><li>[!DNL Excel]檔案，每個重要產品組合設定和重要促銷活動設定都有個別標籤。 資料包含您每個產品組合的產品組合設定，以及不符合最佳實務的每個行銷活動的行銷活動設定。</li></ul> |
| [!UICONTROL Time-of-day Analysis] | （僅適用於具有[!DNL Google Ads]搜尋、顯示或購物行銷活動的產品組合）根據單一產品組合的每小時效能，建議一天中不同時間的[!DNL Google Ads]個行銷活動層級競標修飾詞（廣告排程競標修飾詞）。<br><br>分析包含[!DNL Excel]個活頁簿，其中包含[!UICONTROL Detailed Performance] （依行銷活動時間間隔的績效）和[!UICONTROL Recommended TBAs] （每個行銷活動的建議廣告排程）。 您可以直接將[!UICONTROL Recommended TBAs]工作表匯出至[!DNL Google Ads]編輯器。 分析也包含包含具有支援資料和說明的[!UICONTROL Powerpoint]檔案。 |
