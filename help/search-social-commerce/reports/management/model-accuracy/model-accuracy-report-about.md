# 關於模型精度報表

模型準確度報表可指出用來最佳化出價的成本和收入模型的準確度。

不準確之處可能表示效能問題，因此您可能需要調整投資組合設定。 如需疑難排解預測不準確性的詳細資訊，請參閱「最佳化指南」的「Portfolio」區段中的「關於Portfolio的常見問題集」，此指南可從「搜尋、社交和商務」中取得。<!-- verify convention for referencing Optimization Guide here -->
