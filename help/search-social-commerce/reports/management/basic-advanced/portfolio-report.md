# [!UICONTROL Portfolio Report]

此 [!UICONTROL Portfolio Report] 包含一或多個產品組合的成本、點選次數和（選擇性）轉換資料。 依預設，資料會針對指定日期範圍內的每個時間單位，包含每個適用投資組合的一列，且這些列會先依投資組合遞增排序，然後依時間單位的開始日期遞增排序。

您可以檢視過去36個月的資料。
