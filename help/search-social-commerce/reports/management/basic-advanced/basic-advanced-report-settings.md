| 標籤 | 引數 | 說明 |
|----|----|----|
| 不適用 | [!UICONTROL Name] | （選用）報表和範本的名稱（若您將報表另存為範本）。 如果您套用現有的範本，則預設會填入範本名稱。 如果您未套用範本或輸入名稱，則報表命名為 `<client name>-<date and time>-<report type>` (例如「acme - 2009年4月3日11」:25:上午19:00 PDT — 關鍵字」)（預設）。<br><br>您可以選擇輸入自訂名稱，但不要使用副檔名。<br><br>如果您要建立範本以 [將報表傳送至FTP目錄](/help/search-social-commerce/reports/automation/ftp-reports.md)，您就可以選擇在檔案名稱的任何位置加入「CSV」（大寫字母），以CSV格式（而非預設的TSV格式）建立檔案。 另請參閱 [傳送至FTP目錄之報告的檔案名稱需求](/help/search-social-commerce/reports/automation/ftp-reports.md). |

|  | [!UICONTROL Data Aggregation] | ([!UICONTROL Forecast Accuracy Report only])報表每一列應呈現的時間單位： <i>[!UICONTROL Summary (the default)]</i>， <i>[!UICONTROL Daily]</i>， <i>[!UICONTROL Weekly]</i> （然後指定一週的第一天）， <i>[!UICONTROL Monthly]</i>， <i>[!UICONTROL Quarterly]</i>， <i>[!UICONTROL Yearly]</i>，或 <i>[!UICONTROL Day of Week]</i> （僅限基本報表）。<br><br>報表會針對指定日期範圍內的每個時間單位，為每個關鍵字或其他專案顯示一列。 例如，如果您選取 <i>[!UICONTROL Summary]</i> 和日期範圍 <i>[!UICONTROL Last 7 Days]</i> 對於關鍵字報表，該報表的每個關鍵字各包含一列，每個列彙總七天內的總計和平均值。 相同的報表，具有 <i>[!UICONTROL Daily]</i> 資料彙總在過去七天的每個天中，每個關鍵字各包含一列（每個關鍵字七列）。<br><br><b>附註：</b><ul><li>每小時彙總資料時，使用&quot;[!UICONTROL Hourly]「或」[!UICONTROL Day of Week (Hourly)]&quot;彙總：</li><ul><li>您最多可以檢視過去14天的資料。 報表會以24小時制依小時顯示資料。 &quot;[!UICONTROL 0]&quot;包含00:01-00:59 (12:01-12:上午59點)和&quot;[!UICONTROL 23]「包含23個:01-23:59 (11:01-11:下午59點)。</li><li>「[!UICONTROL Device]「 」欄不受支援。</li><li>對於 [!UICONTROL Ad Group Report]， &quot;[!UICONTROL Data Based On]「選項」[!UICONTROL Portfolio Configuration During the Specified Dates]「」不受支援。</li><li>對於 [!UICONTROL Ad Group Report]，您只能以目前的產品組合設定作為每小時資料的基礎(使用「[!UICONTROL Data Based On]「選項」[!UICONTROL Current Portfolio Configuration]「)，而不是產品組合在指定日期期間的設定。</li></ul></ul> |
|  | [!UICONTROL By Keyword Classification] &amp;vert； [!UICONTROL By Ad Classification] | ([!UICONTROL Label Classification Report] 僅限)是否依關鍵字上的標籤分類或廣告上的標籤分類彙總資料。 |
|  | [!UICONTROL Conversions Based on] | 如何報告轉換資料：<ul><li><i>[!UICONTROL Transaction date]</i> （預設值）：檢視交易日期在指定期間內發生的交易。 此選項會顯示指定期間內已賺取的收入。</li><li><i>[!UICONTROL Click date]：</i> 檢視在指定時段內因點按而發生的交易。 當產品組合在點按和交易之間有重大延遲時，此選項可用於計算產品組合的每次點按歷史收入，其指示一段時間內預期的收入行為。</li></ul> |
|  | [!UICONTROL Date Range] | 要產生資料的日期範圍：<ul><li><i>[預設範圍]：</i> 常見的時間增量清單，範圍從 <i>[!UICONTROL Today]</i> 至 <i>[!UICONTROL Last 180 Days]</i>. 預設值為 <i>[!UICONTROL Last 7 Days]</i>，會報告資料可用的過去七天。 <b>注意：</b> <i>[!UICONTROL Last Month]</i>， <i>[!UICONTROL Last 3 Months]</i>、和 <i>[!UICONTROL Last 6 Months]</i> 顯示前一個日曆月的資料。</li><li><i>[!UICONTROL Custom Date Range]：</i> 指定開始日期和結束日期。 以YYYY/MM/DD或M/D/YYYY格式輸入日期，或按一下 ![行事曆](/help/search-social-commerce/assets/calendar.png "行事曆") 欄位旁邊，並選取日期。</li></ul> |
|  | [!UICONTROL Compare With] | 比較指定日期範圍的資料與第二日期範圍的資料。 選取此選項時，會為每個一般資料欄新增兩個額外的欄。 例如，此報告不只包括「曝光數」的一欄，而包括「」的欄[!UICONTROL Impressions Range 1]，&quot; &quot;[!UICONTROL Impressions Range 2]，」和「[!UICONTROL Impressions Difference].」<br><br><b>附註：</b><ul><li>自訂/衍生量度的差異欄不會顯示。</li><li>比較大型日期範圍的報表需要較長時間才能產生。</li></ul> |
|  | [!UICONTROL Show Comparison Data as] | (當&quot;[!UICONTROL Compare With]」已選取)如何表示「中兩個選定日期範圍內的資料差異」[資料欄位] [!UICONTROL Difference]「欄：<ul><li><i>[!UICONTROL Variance]</i> （預設）：以數值顯示差異。</li><li><i>[!UICONTROL % Change]：</i> 以百分比顯示差異。</li></ul> |
|  | [!UICONTROL Data Based On] | （部分報表型別；僅適用於依投資組合篩選資料時）如何報告行銷活動資料：<ul><li><i>[!UICONTROL Current portfolio configuration]</i> （預設）：顯示目前對應至指定產品組合之行銷活動的資料。 此報表不包含日期範圍內位於產品組合中，但仍不存在的行銷活動的資料。</li><li><i>[!UICONTROL Portfolio configuration during the specified dates]：</i> 當行銷活動對應至指定的產品組合時，顯示行銷活動的資料。 結果可能包括在指定日期範圍內任何時間不在產品組合中，但位於產品組合中的行銷活動資料。 選取此選項時，無法依廣告網路篩選報表。 對於 [!UICONTROL Ad Group Report]，當您使用「 」每小時彙總資料時，不支援此選項[!UICONTROL Hourly]「或」[!UICONTROL Day of Week (Hourly)]&quot;彙總。</li></ul><b>注意：</b> 當您依廣告網路篩選報表時，報表會顯示指定日期範圍內之指定廣告網路帳戶中的促銷活動資料，無論其是否曾對應至產品組合。 |
|  | [!UICONTROL Search/Content] | ([!UICONTROL Basic Reports] 除了 [!UICONTROL Label Classification Report])要報告的管道型別：<ul><li><i>[!UICONTROL Separate sections for each]：</i> 同時包含搜尋和內容/顯示管道的資料，但以管道型別標示的不同列顯示每個資料。</li><li><i>[!UICONTROL Combined]：</i> 結合搜尋和內容管道的資料。</li><li><i>[!UICONTROL Search only]：</i> （僅適用於廣告網路帳戶）僅包含搜尋管道的資料。</li><li><i>[!UICONTROL Content only]：</i> 僅包含來自內容管道的資料，包括顯示和社交行銷活動的所有資料。</li></ul> |
|  | [!UICONTROL Filter By] | （最基本和進階的報表）是否針對特定產品組合或特定廣告網路的資料產生報表：<ul><li><i>[!UICONTROL Portfolio]</i> （預設）：將行銷活動的資料包含在一或多個產品組合中。</li><li><i>[!UICONTROL Search Engine]：</i> 如適用於報表型別，將促銷活動的資料包含在一或多個廣告網路中。 |
|  | \[主要篩選器\] | 要包含的資料元件。 如果您未進行選取，則所有產品組合（如適用）或所有適用廣告網路及其子元件的資料都會納入報告中。 您可選擇指定個別元件和子元件，以縮小要報告的資料範圍。 根據報表型別以及（若適用）您是依產品組合或廣告網路進行篩選，請指定要包含的元件：<ul><li><i>[!UICONTROL Portfolio]：</i> 一或多個產品組合或其子元件（行銷活動或廣告群組）。 若要選取元件及其所有子元件，請選取元件名稱旁的核取方塊。 若要選取子元件，請選取子元件名稱旁的核取方塊，然後按一下>>將其移動到 [!UICONTROL Selected Filters] 欄。 例如，若要取得Portfolio1及其所有促銷活動和廣告群組的資料，請選取Portfolio1旁的核取方塊。 若要在Portfolio1的Campaign 1中發生至少一個事件時取得資料，請展開Portfolio1，然後只選取Campaign 1旁的核取方塊。</li><li><i>[!UICONTROL Search Engine]：</i> 一或多個廣告網路或其子元件（帳戶、促銷活動或廣告群組）。 若要選取元件及其所有子元件，請選取元件名稱旁的核取方塊，然後按一下「>>」將其移至 [!UICONTROL Selected Filters] 欄。 若要選取子元件，請選取子元件名稱旁的核取方塊，然後按一下>>將其移動到 [!UICONTROL Selected Filters] 欄。 例如，若要在任一事件中至少發生一次時取得資料 [!DNL Google Ads] 帳戶、行銷活動和廣告群組，選取旁邊的核取方塊 [!DNL Google AdWords]. 只為中的行銷活動1取得資料 [!DNL Google Ads] 帳戶1，展開 [!DNL Google Ads] 然後選取「帳戶1」，接著僅選取「促銷活動1」旁的核取方塊。</li></ul><b>附註：</b><ul><li>若要展開清單中的元件（例如列出廣告網路上的帳戶），請按一下 ![向右箭號圖示](/help/search-social-commerce/assets/compressed-item.png "向右箭號圖示") 位於元件旁。</li><li>若要檢視任何專案的元件型別，請將游標停留在它上。</li><li>依預設，僅會列出a)作用中及最佳化的產品組合及其作用中元件，或b)作用中及啟用的廣告網路帳戶、行銷活動及其作用中元件。 若要檢視暫停和刪除的元件，請按一下 ![向下鍵](/help/search-social-commerce/assets/arrow-down-expand.png "向下鍵") 旁邊 **[!UICONTROL Show]** 並選取 **[!UICONTROL All]**.</li><li>當您依產品組合產生進階報表時，產生的資料會用於目前對應至指定產品組合的行銷活動。 此報表不包含日期範圍內位於產品組合中，但仍不存在的行銷活動的資料。</li></ul> |
| [!UICONTROL Columns] | [!UICONTROL Use revenue and derived metrics from] | (所有基本和進階報表，但不包括 [!UICONTROL Ad Variation Report]， [!UICONTROL Keyword Report]、和 [!UICONTROL Product Group Report])在報表中填入下列其中一個預設檢視集（也就是使用最小公分母）共有的所有收入和自訂（衍生）量度欄：<ul><li><i>全部：</i> 若要包含內所有標籤的預設檢視所共用的欄超集 [!UICONTROL Portfolios] 和 [!UICONTROL Campaigns] 檢視，如適用於廣告商的產品設定。</li><li><i>[!UICONTROL Portfolios]：</i> 若要包含預設檢視共用的欄，以供 [!UICONTROL Portfolios] 檢視。</li><li><i>[!UICONTROL Search]：</i> 若要包含預設檢視共用的欄，以供 [!UICONTROL Campaigns] 檢視。</li><li><i>[!UICONTROL Display]：</i> 已過時</li><li><i>[!UICONTROL Social]：</i> 已過時</li><ul><b>附註：</b><ul><li>您可以選擇移除或重新排列任何可用的收入或衍生量度欄。</li><li>雖然您可以新增更多欄，但屬性欄是預先為報表所定義。</li><li>請確定您選擇的欄位集適用於您在上選取的廣告網路 [!UICONTROL Basic Settings] 標籤。</li><li>若要變更預設檢視中的欄，請參閱&quot;[編輯預設或自訂檢視](/help/search-social-commerce/common-tasks/data-views/custom-default-views-manage.md).」 您可以選擇將設定儲存到相同頂層檢視中的所有其他標籤。</li></ul> |
