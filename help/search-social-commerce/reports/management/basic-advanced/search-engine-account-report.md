# [!UICONTROL Search Engine Account Report]

此 [!UICONTROL Search Engine Account Report] 包含一或多個廣告網路帳戶與行銷活動的成本、點選和（選擇性）轉換資料。 依預設，資料會針對指定日期範圍內每個時間單位的每個適用帳戶包含一個資料列，且這些資料列會先依廣告網路，再依帳戶遞增排序。

您可以檢視過去36個月的資料。
