# [!UICONTROL Search Engine Account Report]

此 [!UICONTROL Search Engine Account Report] 包含一或多個具有行銷活動的廣告網路帳戶的成本、點選和（選擇性）轉換資料。 依預設，資料在指定日期範圍內每個時間單位都包含每個適用帳戶的一列，且這些列會先依廣告網路，然後依帳戶遞增排列。

您可以檢視過去36個月的資料。
