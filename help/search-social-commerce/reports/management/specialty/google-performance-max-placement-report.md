# [!UICONTROL Google Performance Max Placement Report]

*僅適用於[!DNL Google Ads]個具有最高成效行銷活動的帳戶*

[!UICONTROL Google Performance Max Placement Report]包含指定帳戶中[!DNL Google Ads]個最高成效行銷活動中的位置曝光資料。 報表包含依目標URL的每個包含位置的曝光數。 根據預設，對於在指定資料範圍內至少收到一次曝光的每個位置和目標URL組合，資料會包含一列，即使位置在報告期間被刪除也一樣。 依預設，這些列會依促銷活動以遞增順序排列。

使用此報表來尋找要排除的URL，並改善最高成效行銷活動的效能。
