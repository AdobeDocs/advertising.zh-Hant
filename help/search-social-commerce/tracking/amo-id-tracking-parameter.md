# AMO ID (s_kwcid)追蹤引數

*僅整合Adobe Advertising-Adobe Analytics的廣告商*

Adobe Advertising會使用AMO ID附加引數（也稱為`s_kwcid`引數），與Adobe Analytics共用行銷活動的相關資料，此引數包含廣告頻道和廣告網路專用元素。

如需如何將AMO ID引數新增至您的追蹤URL，以及DSP和Search、Social與Commerce的AMO ID格式的詳細資訊，請參閱「[由 [!DNL Analytics]](/help/integrations/analytics/ids.md#amo-id)使用的Adobe Advertising ID」。
