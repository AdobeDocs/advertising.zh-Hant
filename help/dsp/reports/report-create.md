# 建立自訂報表

1. 在主功能表中，按一下&#x200B;**[!UICONTROL Reports]**。
1. 按一下&#x200B;**[!UICONTROL New Custom Report]**。
1. 指定[報表設定](/help/dsp/reports/report-settings.md)。
1. 按一下&#x200B;**[!UICONTROL Create Custom Report]**。
