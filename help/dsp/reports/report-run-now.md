# 運行自定義報告

>[!NOTE]
>
>您還可以在 [建立](report-create.md) 或 [編輯](report-edit.md) 它。

1. 在主菜單中，按一下 **[!UICONTROL Reports]**。

1. 在 [!UICONTROL Options] 列，按一下 **[!UICONTROL options]** > **[!UICONTROL Run Now]**。

1. 在確認消息中，按一下 **[!UICONTROL OK]**。

   完成報告後，將其發送到報告設定中指定的所有電子郵件地址。
