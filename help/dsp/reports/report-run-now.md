# 執行自訂報表

您可以執行尚未過期且目前未執行的任何報告。

>[!NOTE]
>
>您也可以在您[建立](report-create.md)或[編輯](report-edit.md)自訂報表時執行它。

1. 在主功能表中，按一下&#x200B;**[!UICONTROL Reports]** > **[!UICONTROL Custom Reports]**。

1. 在報告名稱旁邊，按一下「**[!UICONTROL ...]** > **[!UICONTROL Run Now]**」。

   報表完成時，會傳送至報表設定中指定的目的地。
