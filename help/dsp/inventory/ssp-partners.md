# SSP合作夥伴

供應端平台(SSP)是一種廣告技術平台，用於協助數位媒體擁有者和發行者在自動化拍賣中銷售數位廣告。 數位媒體擁有者和發佈者使用SSP來協調、管理其廣告詳細目錄的供應和分發，並從中獲利。 Advertising DSP已整合所有領先的SSP，讓廣告商可輕鬆存取符合其廣告需求的高品質詳細目錄。

>[!NOTE]
>
>所有SSP合作關係可能會有所變更。 此清單反映目前的合作關係。
