# SSP 合作夥伴

供應端平台(SSP)是一種廣告技術平台，用於協助數位媒體擁有者和發行者在自動化拍賣中銷售數位廣告。 数字媒體擁有者和发布者使用 SSP 來協調、管理和創造營收其廣告詳細目錄的供應和分發。 廣告DSP與所有領先的 SSP 集成，使廣告商能夠輕鬆訪問高品質的廣告資源以滿足其廣告需求。

>[!NOTE]
>
>所有 SSP 合作夥伴關係都可能發生變化。 這一清單反映了目前的夥伴關係。
