# 使用AI輔助聊天介面搜尋產品檔案

*Beta功能*

*僅支援英文*

<!-- How will this work once we have unified shell, which has its own version of AI Assistant? -->

使用AI聊天介面在[Advertising DSP指南](/help/dsp/home.md)和（使用Advertising Creative的廣告商） [Advertising Creative指南](/help/creative/home.md)中搜尋概念和操作說明內容。 答案僅以[Experience League](https://experienceleague.adobe.com/zh-hant/docs/advertising)中針對這些產品所記載的內容為基礎。

回應包括引文以及其他提示和後續問題，可幫助您縮小查詢範圍並尋找更多資訊。 您的聊天記錄會保留，您的查詢不會與其他使用者共用。

>[!NOTE]
>
>查詢不會傳回有關您帳戶的資料，例如行銷活動、交易或對象設定、狀態或效能。 它們也無法協助您疑難排解問題。

![查詢和回應的範例](/help/dsp/assets/agent-chat-response.png "查詢和回應的範例")

>[!IMPORTANT]
>
>AI產生的回應可能不準確或誤導。 一律先驗證回應和來源，再將其用於影響成本或投入的決策。

## 範例查詢

您可以詢問有關行銷活動管理、最佳化、對象管理、交易、報告和其他產品功能的問題。 以下是範例：

* 如何將廣告附加至投放位置？

* 在刊登版位設定中使用不同步調選項有何後果？

* 何時應該使用每種型別的最佳化目標？

* 程式化預留(PG)版位為何不提供曝光數？

* 哪些報表包含住戶層級資料？

* 在[!DNL Creative]中，目標體驗與非目標體驗有何不同？

* 如何為[!DNL Creative]體驗建立廣告標籤？

## 提交查詢

您可以在一則訊息中詢問多個問題，但一次只能詢問一則訊息。 請等待回應，然後再傳送其他回應。

1. 按一下任何頁面右上角的![代理聊天](/help/dsp/assets/agent-chat.png "代理聊天")。

1. 輸入您的查詢並按一下![提交提示](/help/dsp/assets/submit-prompt.png "提交提示")。

   如需詳細資訊，請參閱[正在寫入提示](#writing-prompts)。

   回應包含內嵌引文和底部的&#x200B;**[!UICONTROL Documentation Sources]**&#x200B;清單。 後續問題和建議也會出現。

1. （可選）若要開啟用作資料來源的頁面，請執行下列其中一項作業：

   * 按一下編號的引文。

   * 按一下&#x200B;**[!UICONTROL Documentation Sources]**&#x200B;以顯示回應中引用的所有頁面清單，然後按一下頁面連結。

## 提供回應的相關意見

<!-- Not actionable in terms of improving page content -->

* 在[!UICONTROL Documentation Sources]清單旁：

   * 如需有用的回應，請按一下![向上縮圖](/help/dsp/assets/thumbs-up.png "向上縮圖")。

   * 如需無用的回應，請按一下![向下縮圖](/help/dsp/assets/thumbs-down.png "向下縮圖")。

## 寫入提示的基本知識 {#writing-prompts}

* **清楚明確。**&#x200B;使用完整問題（「如何訂閱隨選詳細目錄？」）、任務片語（「訂閱隨選詳細目錄」）或主題片語（「隨選詳細目錄」）。

* 儘可能與產品功能（例如「行銷活動」或「交易」）的UI辭彙&#x200B;**相符**。

  對於常見動作（例如「新增」、「附加」或「指派」），您可以使用同義字。

* **新增詳細資料**&#x200B;以縮小或擴大結果。 視需要完善後續問題。

* **不必擔心標點符號與大寫**，除非它們會影響意義。
