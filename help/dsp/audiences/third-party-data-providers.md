# Advertising DSP中可用的第三方資料提供者

使用每個區段的費用，會顯示在位置設定和儲存的對象設定中的區段名稱旁。

* [!DNL digitalAudience]
* [!DNL eXelate] ([!DNL Nielsen])

* [!DNL Fraud Protection]

* [!DNL LiveRamp]
* [!DNL Lotame]
* [!DNL NinthDecimal]
* [!DNL Semcasting]
* [!DNL TailTarget]
* [!DNL Tapfwd]
* [!DNL TransUnion]
* [!DNL TruSignal]
* [!DNL Weborama]
* [!DNL Zeotap]
