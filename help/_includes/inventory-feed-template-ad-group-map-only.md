# 文字廣告範本 — 僅廣告群組對應

**[!UICONTROL Map Only]：** （除[!DNL Yandex]以外的所有廣告網路；選擇性）將所有新關鍵字和廣告對應至現有的廣告群組，而非建立新的廣告群組。 如果啟用此選項，請選取對應方法。

在廣告群組層級使用[!UICONTROL Map Only]需要現有的帳戶結構，此結構與產品分類法緊密相連結，並且可輕鬆對應至資料摘要。
