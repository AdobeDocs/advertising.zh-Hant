# 編碼帳戶和促銷活動設定中的基底URL欄位

**[!UICONTROL Encode Base URL]：** （只有具有目的地URL和追蹤型別[!UICONTROL EF Redirect]的帳戶）一般使用者瀏覽器位址列中的URL是否包含字元編碼（例如`%3D`而非`=`）：

* *[!UICONTROL On]* （預設值）：顯示URL中的字元編碼。

* *[!UICONTROL Off]：*&#x200B;顯示未編碼的URL。
