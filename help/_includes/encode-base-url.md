# 編碼帳戶和行銷活動設定中的基本URL欄位

**[!UICONTROL Encode Base URL]：** (具有目的地URL和追蹤型別的帳戶 [!UICONTROL EF Redirect] 僅限)一般使用者是否會看到字元編碼(例如 `%3D` 而非 `=`)，網址列中的URL：

* *[!UICONTROL On]* （預設）：顯示URL中的字元編碼。

* *[!UICONTROL Off]：* 顯示未編碼的URL。
