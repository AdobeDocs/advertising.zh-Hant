# 文字廣告範本 — 範本名稱

**[!UICONTROL Template Name]：**&#x200B;範本的名稱。 您可以變更現有範本的名稱，而不會產生任何後果。 範本名稱不能包含下列任何字元： `# % * | \ : " < >  . ? /` （允許`&`）

若要使用與現有範本相同的引數建立新範本，請複製範本並為其指定新名稱。 從新範本建立的廣告不會與從來源範本建立的廣告相關聯。
